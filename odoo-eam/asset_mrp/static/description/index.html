Integrate Asset and Manufacturing

Integrate manufacturing and maintenance asset management.

This module allows use the same Assets for manufacturing and maintenance purposes. Integration take in account following assumption. In manufacturing, Work Center can be simple logical entity, but also can reference to equipment that is physical asset. Each physical asset must be maintenable, but not each can be manufacturing equipment. So, when you create Work Center, you can reference it to asset. Keep one entity in one place for escape mistakes!
